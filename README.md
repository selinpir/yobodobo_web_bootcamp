# Kişisel Portfolyo Sitesi

Bu proje, Web Build Bootcamp kapsamında verilen **Kişisel Portfolyo Sitesini Geliştirme** ödevi için hazırlanmıştır. Projede temel HTML yapısı geliştirilmiş, eksik CSS alanları tamamlanmış ve sayfaya JavaScript etkileşimleri eklenmiştir.

## Proje Amacı

Bu çalışmanın amacı, hazır iskeleti verilen kişisel portfolyo sitesini daha düzenli, modern, responsive ve etkileşimli bir hale getirmektir.

## İçerik

Projede aşağıdaki bölümler bulunmaktadır:

- Ana Sayfa
- Portföy
- Eğitimlerim
- Deneyimlerim
- Becerilerim
- Projelerim
- Sosyal Medya
- Yorumlar
- Footer

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- Font Awesome

## Eklenen CSS Geliştirmeleri

Ödev kapsamında aşağıdaki CSS alanları düzenlenmiş ve geliştirilmiştir:

- `project-container`
- `project-card`
- `overlay`
- `comment-container`
- `comment-card`
- `card-description`
- `footer`

Ayrıca şu görsel geliştirmeler yapılmıştır:

- Responsive yapı
- Hover efektleri
- Kart hizalama ve boşluk düzeni
- Mobil görünüm uyumluluğu
- Dark mode uyumu

## Eklenen JavaScript Özellikleri

Projeye en az 3 farklı JavaScript etkileşimi eklenmiştir:

- Navbar linklerine tıklayınca yumuşak kaydırma
- Sayfa aşağı indikçe aktif menü başlığının değişmesi
- “Yukarı çık” butonu
- Dark mode / light mode butonu
